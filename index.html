History

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
Git is easy to learn and has a tiny footprint with lightning fast performance. It outclasses SCM tools like Subversion, CVS, Perforce, and ClearCase with features like cheap local branching, convenient staging areas, and multiple workflows.
Learn Git in your browser for free with Try Git.

Starter

  • git init : Initialize a directory with git
  • git status: Give to the user status of the content
  • git add .: Put content in a transit zone
  • git commit -m "commit name": Give a version online
  • git push: Push commited content on github
  • git branch: Display branches in ready to push
  • git branch "name": create a local branch
  • git checkout "branch name": Move user in an another branch
  • git log: Show all commit to the user

Move Up !

Advanced :

  • git rm file: Remove file
  • git rm dir/ -r: Remove file
  • git clone existing_dir new_dir: Create from other repository
  • git remote add path/url : Add remote
  • gh-pages: Tool from github to generate a weblink
